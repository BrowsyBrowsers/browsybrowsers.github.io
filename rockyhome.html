🏡 RockyHome

Launchpad, UI shell, and dark mode environment within RockyBrowser

🌑 Native Dark Mode

RockyHome inherits shell-level theming and adapts to overlays defined by tabd.plugin.

📁 Modular Environment

Acts as the primary canvas for RockyShell overlays, bookmarks, and visual stubs.

📦 Extension-Friendly UI

While tabd.plugin lives in RockyBrowser, RockyHome supports its rendering logic and shell triggers.
